This program is uses simple function, this is a simple Registration and Login System 
that uses java.io Package in creating .file that holds data. Like username and password.

How to use:
you have open this program to your IDE then run it..

Featured function in this project:
*Registration(with validation).
*Login(with validation).
*Dashboard(you can add, edit, save, delete and search data).

Note:
> the program will create "./Database.file" in directory, that contains Username and Password you made.
